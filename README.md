## 📖 Sobre o projeto

O **Convert Money** é uma aplicação web desenvolvida para realizar conversões de valores em reais para outras moedas de maneira rápida e simples.

O usuário informa um valor em real, escolhe a moeda desejada e recebe o resultado formatado automaticamente.

O projeto utiliza uma API de cotações para consultar valores atualizados do dólar e do euro, proporcionando resultados mais próximos do mercado.

## ⚙️ Como funciona

O usuário digita o valor que deseja converter e seleciona uma das moedas disponíveis.

Ao clicar no botão de conversão, a aplicação consulta a cotação atual, realiza o cálculo e exibe o resultado no formato correspondente à moeda escolhida.

```text
Valor em Real
      ↓
Seleção da moeda
      ↓
Consulta da cotação pela API
      ↓
Cálculo da conversão
      ↓
Formatação e exibição do resultado
```

## 🔌 Integração com API

A aplicação utiliza a AwesomeAPI para consultar as cotações do dólar e do euro em relação ao real.

A consulta é realizada por meio da Fetch API do JavaScript, permitindo que os valores sejam atualizados diretamente pela fonte externa.

## ⚠️ Observação

Os valores possuem finalidade demonstrativa e podem apresentar pequenas diferenças em relação às taxas praticadas por bancos, corretoras e casas de câmbio.
